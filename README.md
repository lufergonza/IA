# Inteligencia Artificial - Universidad del Pacífico

## Descripción

Este repositorio contiene las actividades, talleres, proyectos, investigaciones y ejercicios desarrollados durante el curso de Inteligencia Artificial de la Universidad del Pacífico.

El propósito de este repositorio es almacenar de forma organizada el material académico trabajado a lo largo del semestre, permitiendo llevar un registro de las actividades realizadas y los conocimientos adquiridos en el área de Inteligencia Artificial.

## Contenido del repositorio

- Primer Corte Académico
- Segundo Corte Académico
- Tercer Corte Académico
- Talleres y actividades prácticas
- Investigaciones
- Notebooks y ejercicios en Python
- Proyectos relacionados con Inteligencia Artificial

## Tecnologías utilizadas

- Python
- Pandas
- NumPy
- Jupyter Notebook
- Google Colab
- Git y GitHub
