
# Análisis Estadístico Descriptivo: Comportamiento de Descargas de Apps 📊📱



## 📌 Descripción del Proyecto
Este repositorio contiene el desarrollo del **Taller No. 3 de Análisis Estadístico Descriptivo y Visualización de Datos**, realizado como parte de la asignatura de Inteligencia Artificial (Semestre 8 - Ingeniería de Sistemas). 

El proyecto se centra en el análisis cuantitativo de una muestra de 50 usuarios para evaluar su comportamiento respecto a la cantidad de aplicaciones móviles descargadas. El objetivo principal es transformar datos brutos en información útil para la toma de decisiones tecnológicas y estratégicas.

## 🎯 Objetivos
* Identificar variables y características clave de un conjunto de datos poblacional.
* Construir una **Tabla de Distribución de Frecuencias** aplicando la Regla de Sturges.
* Diseñar representaciones gráficas precisas (**Histograma, Polígono de Frecuencias y Ojiva**).
* Analizar el comportamiento de los datos mediante evidencia cuantitativa para extraer conclusiones funcionales sobre los límites de almacenamiento y uso de los usuarios.

## 📂 Estructura del Repositorio
* `data/`: Contiene el conjunto de datos original suministrado en formato `.xlsx`.
* `informe/`: Incluye el documento técnico final en formato PDF con los cálculos estadísticos completos y la interpretación de los gráficos.
* `images/`: Capturas de las visualizaciones de datos generadas (Histograma, Polígono, Ojiva).

## 🚀 Resultados y Conclusiones Clave
A partir del análisis descriptivo de la variable *Apps Descargadas*, se determinó que:
1.  **Tendencia Central:** La mayor concentración de usuarios (20%) posee entre 60 y 74 aplicaciones instaladas, consolidándose como la categoría modal.
2.  **Límites de Saturación:** El 80% de la población evaluada mantiene sus dispositivos con menos de 105 aplicaciones, lo que demuestra un comportamiento de equilibrio funcional estandarizado.
3.  **Distribución:** Los datos presentan una asimetría positiva (sesgo a la derecha), indicando que el consumo masivo de almacenamiento (más de 120 apps) es estadísticamente excepcional (solo el 10% de la muestra).

## 🛠️ Herramientas y Tecnologías
* **Análisis de Datos:** Microsoft Excel / Herramientas de hojas de cálculo.
* **Metodologías:** Estadística Descriptiva, Regla de Sturges, Análisis de Frecuencias Relativas Acumuladas.
* **Documentación:** Markdown, Microsoft Word/PDF.
